# API-ETL
Uma API RESTful que executa pipelines de folha de pagamento em PDF e planilhas de vendas em XLSX.

## Stack atual

- Python 3.12 nos containers Docker
- FastAPI para a API
- PostgreSQL para vendas e DynamoDB para custos de funcionários
